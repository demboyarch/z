Quick Start Guide

Welcome to Z! This quick start guide will walk you through the basic features and get you up and running with Z in no time.

Note: This guide assumes you've already installed Z on your system.

The Welcome Screen

When you first open Z, you'll see the welcome screen with these main options:

  • New Project: Create a new coding project
  • Open Project: Open an existing local project folder
  • Clone Repository: Clone a Git repository to work on

The welcome screen also displays the current version of Z and provides access to settings like analytics and language preferences.

Changing the Interface Language

Z supports multiple languages through the Z-Rosetta system. To change the interface language:

  1. Look for the language selector dropdown at the bottom of the welcome screen
  2. Click it and select your preferred language
  3. The interface will immediately update to reflect your selection

Your language preference is automatically saved for future sessions. See the Z-Rosetta documentation for more details.

Creating Your First Project

Option 1: New Project

  1. Click New Project on the welcome screen
  2. Select a template or start with an empty project
  3. Choose a location to save your project
  4. Give your project a name
  5. Click Create Project

Option 2: Open Existing Project

  1. Click Open Project on the welcome screen
  2. Browse to select a folder containing your project
  3. Click Open

Option 3: Clone a Git Repository

  1. Click Clone Repository on the welcome screen
  2. Enter the repository URL (HTTPS or SSH)
  3. Choose a local destination folder
  4. Click Clone

Understanding the Z Interface

Once you've opened a project, you'll see the main Z interface with these components:

1. Editor Window

The central area where you edit your code. Z features:

  • Syntax highlighting for various languages
  • Intelligent code completion
  • Multiple cursor support
  • Inline error and warning indicators

2. File Explorer

Located on the left side, showing your project's file structure:

  • Navigate through project folders
  • Create, rename, and delete files
  • Right-click for additional file operations

3. Status Bar

At the bottom of the window, providing information like:

  • Current file path
  • Line and column position
  • File encoding and line ending type
  • Current Git branch (if applicable)
