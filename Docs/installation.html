Installing Z

This guide will help you install Z on your system. Z is available for Windows, macOS, and Linux operating systems.

Note: Z is currently in development. While it's stable for everyday use, you may encounter occasional bugs.

System Requirements

Operating System Minimum Requirements Recommended
Windows Windows 10, 4GB RAM Windows 10/11, 8GB RAM
macOS macOS 10.15 (Catalina), 4GB RAM macOS 12+ (Monterey), 8GB RAM
Linux Ubuntu 20.04 or equivalent, 4GB RAM Ubuntu 22.04 or equivalent, 8GB RAM

Windows Installation

Using the Installer

  1. Download the latest Z installer from the releases page.
  2. Run the Z-Setup-x.y.z.exe file.
  3. Follow the installation wizard instructions.
  4. Launch Z from the Start menu or desktop shortcut.

Portable Version

If you prefer a portable installation:

  1. Download the Z-x.y.z-win-portable.zip from the releases page.
  2. Extract the ZIP file to your desired location.
  3. Run Z.exe to start the editor.

macOS Installation

Using the DMG

  1. Download the latest Z-x.y.z.dmg file from the releases page.
  2. Open the DMG file.
  3. Drag Z to the Applications folder.
  4. Launch Z from Applications or using Spotlight.
Note for macOS users: When opening Z for the first time, you may need to right-click the app and select "Open" due to security settings.

Linux Installation

Using AppImage

The AppImage format is supported on most Linux distributions:

  1. Download the Z-x.y.z.AppImage file from the releases page.
  2. Make it executable: chmod +x Z-x.y.z.AppImage
  3. Run the AppImage: ./Z-x.y.z.AppImage

Debian/Ubuntu (DEB package)

  1. Download the z_x.y.z_amd64.deb file.
  2. Install using: sudo dpkg -i z_x.y.z_amd64.deb
  3. Launch Z from your applications menu or using the z command.

Fedora/RHEL (RPM package)

  1. Download the z-x.y.z.x86_64.rpm file.
  2. Install using: sudo rpm -i z-x.y.z.x86_64.rpm
  3. Launch Z from your applications menu or using the z command.

Building from Source

If you prefer to build Z from source:

  1. Clone the repository: git clone https://github.com/z-editor/z.git
  2. Navigate to the directory: cd z
  3. Install dependencies: npm install
  4. Build the application: npm run build
  5. Run Z: npm start

Verifying Your Installation

After installing Z, you should see the welcome screen when you first open the application. The welcome screen includes:

  • Options to create a new project, open an existing project, or clone a repository
  • The current version number and build information
  • The language selector (powered by Z-Rosetta)

Troubleshooting

Common Installation Issues

Windows

  • Application doesn't start: Ensure you have the latest Visual C++ Redistributable installed.
  • SmartScreen warning: Select "More info" then "Run anyway" if you trust the source.

macOS

  • "App is damaged" message: Right-click the app, select "Open" from the context menu, then confirm.
  • Permission issues: Make sure you have proper permissions for the Applications folder.

Linux

  • Missing dependencies: Install required packages using your distribution's package manager.
  • AppImage not starting: Check the permissions and ensure it's executable.

Getting Help

If you encounter issues installing Z:

  • Check the GitHub issues to see if your problem has been reported
  • Join our community Discord for real-time support
  • File a new issue on GitHub with details about your system and the error you're experiencing
