API Reference

This page provides comprehensive documentation for Z's public APIs and extension points.

Note: The Z API is currently in development. This documentation will be expanded as the API stabilizes.

API Overview

Z provides several APIs that you can use to extend and enhance the editor:

  • IPC API - For communication between main and renderer processes
  • Plugin API - For creating plugins that add new functionality
  • Theme API - For creating custom themes
  • Language API - For adding support for new programming languages

IPC API

Z uses the Electron IPC system to communicate between the main process and renderer processes. The following IPC channels are available:

Window Controls

// Minimize the window
window.windowControls.minimize()

// Maximize/restore the window
window.windowControls.maximize()

// Close the window
window.windowControls.close()

Project Management

// Select a directory
window.project.selectDirectory()

// Open a project
window.project.openProject(projectPath)

// Register event handler for when a project is loaded
window.project.onProjectLoaded(callback)

// Get file tree for a project
window.project.getFileTree(projectPath)

Git Integration

// Clone a repository
window.git.cloneRepository(url, directory)

// Get clone progress updates
window.git.getCloneProgress(callback)

Z-Rosetta API

// Get current language and available languages
window.zRosetta.getLanguage()

// Get translations for current language
window.zRosetta.getTranslations()

// Set the language
window.zRosetta.setLanguage(language)
